# postgre-demodb
PostgreSQL demo database in a docker container

1. Скачать архив базы данных: <br>
   https://edu.postgrespro.ru/demo-medium.zip
2. Распаковать архив в директорию ./docker
3. Запустить init.sh: <br>
```bash init.sh```<br>
